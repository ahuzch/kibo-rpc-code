# Kibo RPC Code

This repository contains implementations for the Kibo Robot Programming Challenge (Kibo-RPC).

## Structure

- **code-using-opencv/**  
  Java-based Kibo RPC code using OpenCV for image detection  

- **code-using-yolo/**  
  Java-based Kibo RPC code using YOLO for image detection  

- **yolo code/**  
  Python-based YOLO implementation
